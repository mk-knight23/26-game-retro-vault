How to Use

Navigation: Click on any app card to open it. Click "Back to Hub" or "Home" menu to return.

Games: Each game has on-screen instructions.

Keyboard Shortcuts:

  • ESC - Return to hub from any app
  • Arrow Keys - Control Snake game
  • 0-9, +, -, *, / - Calculator input
  • Enter - Calculator equals, start games
  • Space - Snake pause/resume

Tips: All high scores are saved automatically!
